Defining color value..

Colors in css most often specifies in the following format:

  1. A COLOR Keyword - like - "red", "green","blue","transparent",
  2. A HEX value like "#ff0000" , "#00f000" , "#00ff"
  3. An RGB value (RED, GREEN, BLUE)- like "rgb(255,0,0)"
